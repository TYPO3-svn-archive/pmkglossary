Common markers:

Markers:
FIRSTCHAR - First character in block of word (Uppercase)
BLOCKID - Id of block
TITLE - Glossary catchword.
ALTTITLE - Glossary catchword in alternate spelling.
BODYTEXT - Description of catchword.
IMAGE - Image.
IMAGEWIDTH - Width of image.
IMAGEORIENT - Image orintation.

Special Markers:
PAGEURL - URL of current page.
GLOSSARY_ID - ID of Glossary Plugin element.
GLOSSARY_MENU - Menu of first letters of Glossary words.
                Only works at the topmost level of the template.
                Menu is defined in seperate template part.

News Language Markers (see pi/locallang.xml):
TO_TOP - Text "To top"
WORDS_STARTING_WITH - Text "Words starting with:"
